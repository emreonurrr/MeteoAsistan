# ✨ Özellikler

- 🌌 Dinamik gündüz / gece atmosfer sistemi
- ☀️ Güneş ve ay geçiş animasyonları
- ⭐ Sinematik yıldız alanı efekti
- 🌡️ Gerçek zamanlı sıcaklık verileri
- 💧 Nem oranı gösterimi
- 💨 Rüzgar hızı analizi
- 🧠 Algoritmik kıyafet öneri sistemi
- 🔊 Sesli öneri okuma sistemi
- 📱 Responsive modern tasarım
- ⚡ Vanilla JavaScript altyapısı
- 🎨 Glassmorphism UI tasarımı
